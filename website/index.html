Vibe Ensemble

Multi-Agent Coordination MCP Server for Claude Code

What is Vibe Ensemble?

Vibe Ensemble is a powerful Model Context Protocol (MCP) server that enables sophisticated multi-agent coordination within Claude Code. It allows one coordinator agent to manage multiple specialized worker agents, preventing context drift and maintaining focus across complex, multi-stage projects.

🎯 Focused Execution

Each worker agent specializes in specific tasks, maintaining deep focus without context dilution.

🔄 Seamless Coordination

Coordinator manages task assignment, progress tracking, and inter-agent communication.

📊 Project Management

Built-in ticketing system with multi-stage execution plans and detailed progress tracking.

🚀 Scalable Architecture

Spawn multiple workers, manage queues, and handle complex workflows with ease.

How It Works

The system consists of three main components:

Coordinator Agent

Regular Claude Code instance connected to the MCP server. Manages overall project strategy and task distribution.

Worker Agents

Specialized Claude Code headless processes. Each focuses on specific stages of work (analysis, implementation, testing, etc.).

MCP Server

Rust-based coordination server that manages workers, queues, and provides tools for multi-agent communication.

curl -fsSL https://get.vibeensemble.dev/install.sh | sh
iwr -useb https://get.vibeensemble.dev/install.ps1 | iex

Usage

After installation, configure Claude Code integration:

1. Generate Claude Code configuration:

vibe-ensemble-mcp --configure-claude-code --host 127.0.0.1 --port 3000

2. Start the server:

vibe-ensemble-mcp --port 3000

3. Open Claude Code and run:

vibe-ensemble

Alternatively, configure manually: claude mcp add --transport http vibe-ensemble http://localhost:3000/mcp

1. Generate Claude Code configuration:

Features

🏗️ Project Management

Create and manage projects with automatic workspace detection and configuration.

🎫 Advanced Ticketing

Multi-stage ticket system with execution plans, worker assignments, and progress tracking.

👥 Worker Management

Spawn, monitor, and coordinate multiple specialized worker agents with different capabilities.

📋 Task Queues

Efficient task distribution with priority queues and automatic load balancing.

📈 Event Tracking

Comprehensive logging and monitoring of all agent activities and system events.

💾 Persistent State

SQLite-based storage ensures all project data, tickets, and progress are preserved.
