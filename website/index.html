Installation

Build from Source (Recommended)

git clone https://github.com/siy/vibe-ensemble-mcp.git
cd vibe-ensemble-mcp
cargo build --release

Prerequisites

# Install Rust 1.80+ first
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Claude Code Setup

1

Start the Server (Optional)

Start the server for web dashboard access (Claude Code will also start it automatically):

./target/release/vibe-ensemble
2

Configure Claude Code

Add Vibe Ensemble to Claude Code using the MCP configuration:

# Option 1: Claude CLI (if available)
claude mcp add vibe-ensemble /path/to/vibe-ensemble-mcp/target/release/vibe-ensemble

# Option 2: Manual JSON config in Claude Code settings
{
  "vibe-ensemble": {
    "command": "/path/to/vibe-ensemble-mcp/target/release/vibe-ensemble",
    "args": []
  }
}
