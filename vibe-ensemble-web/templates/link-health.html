About Link Validation
This dashboard monitors the health of all discovered links in the Vibe Ensemble application, including:
• Page routes (/, /dashboard, /agents, etc.)
• API endpoints (/api/health, /api/stats, etc.)
• WebSocket connections (/ws)
• Authentication pages (/login, /logout)

Status Meanings:
Healthy - Link responds with 2xx status
Warning - Link redirects (3xx) or special handling needed
Broken - Link returns error or times out
